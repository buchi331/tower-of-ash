# 『創世のヒント』(仮) ── 各時代リサーチ集

神様視点で人類を猿人から現代文明へ導くゲームの、史実ベース転換点リサーチ。
各転換点は「岐路 / 授けものの候補 / 諸刃の結果 / IF分岐(世界の歪み) / 影響する軸 / ジレンマ」で構成。
軸 = 〈技術〉〈自然〉〈平等〉〈平和〉〈精神/文化〉。年代は諸説あるものが多い（ゲーム内では「神々でも見解が割れる」演出に流用可）。

---

## 第1幕：人類の黎明（約700万年前 〜 前1万年）

### 1. 直立二足歩行（約700万〜440万年前）
- 岐路: 森が縮小、草原へ。木に残るか地に降りるか。
- 授けもの: [立て(二足歩行)] / [木に残れ] / [四つ足で速く]
- 諸刃: 手が空き視界が広がる→道具・運搬・育児へ。だが産道が狭まり「難産」が人類の宿命に。
- IF: 「木に残れ」=賢い類人猿のまま物語終了(バッドエンドの核)。急ぎ過ぎ=出産死多発の「呪われた繁栄」。
- 軸: 技術+ / 自然- / 精神+
- ジレンマ: 賢さの代償に母が死ぬ確率が永久に上がる。進歩と命の最初の取引。

### 2. 石器の発明（約330万〜176万年前）
- 岐路: 牙も爪もない。硬い死肉・骨髄を開けられない。
- 授けもの: [石を割れ(打製)] / [左右対称に整えよ(握斧)] / [歯と爪で頑張れ]
- 諸刃: 肉食→脳の大型化の好循環。だが同じ刃は「最初の武器」=格差と暴力の起点。
- IF: 対称美に固執=儀式に呑まれた職人文明。与えない=脳が育たず絶滅候補。
- 軸: 技術++ / 平和- / 自然-
- ジレンマ: 食べる刃は殺す刃。豊かさと暴力は同じ石から。

### 3. 火の手なずけ（確実な証拠 約100万年前）
- 岐路: 夜は捕食者の時間、寒さと生食の負担。自然火災の火種を留められるか。
- 授けもの: [火を運べ(管理)] / [火で焼け(調理)] / [火を恐れよ]
- 諸刃: 調理が腸を短く脳を大きく。焚き火=夜のだんらん(物語と社会性)。だが延焼・森林破壊・戦火の遠い起源。
- IF: 畏怖で授ける=火を独占する「灰の神権政治」。乱用=森を焼き食料基盤を燃やす自滅。
- 軸: 技術++ / 精神+ / 自然--
- ジレンマ: 最初の「地球を変える力」を渡してよいか。

### 4. 言葉と象徴的思考（基盤 約30万〜10万年前）
- 岐路: 群れが大きく、情報共有と知識継承の手段が乏しい。
- 授けもの: [声に意味を(言語)] / [印を残せ(記号)] / [身振りで足りる]
- 諸刃: 知識の累積と大集団協力=文明全ての土台。だが嘘・扇動・「我ら/彼ら」も言語が生む。
- IF: 記号だけ=無言で壁画が氾濫する不気味な静寂文明。嘘特化=疑心暗鬼の社会。
- 軸: 精神++ / 技術+ / 平和±
- ジレンマ: 真実を伝える力と嘘で操る力は同じ贈り物。

### 5. 認知革命・装身具（約10万〜7万年前, アフリカ先行）
- 岐路: 体は現生人類だが暮らしは平凡。自己と所属を表す術がない。
- 授けもの: [身を飾れ(貝ビーズ)] / [印を刻め(幾何学文様)] / [飾りは無用]
- 諸刃: 交易網・美意識・自己表現。だが「飾り=持つ者/持たざる者」最初の格差表現。
- IF: 過剰=貝の数で人の価値が決まる虚飾社会。皆無=賢いが無文化のまま眠る。
- 軸: 精神++ / 技術+ / 平等-
- ジレンマ: 「自分らしさ」は誇りと差別の両刃。

### 6. 死者を葬る（約10万〜12万年前）
- 岐路: 仲間の死。放置は病と捕食者を招く。喪失と「あの人はどこへ?」の感情。
- 授けもの: [土に還し印を添えよ(副葬埋葬)] / [清く処理せよ(衛生)] / [忘れて去れ(遺棄)]
- 諸刃: 来世観=宗教・倫理・共同体の絆。だが死の恐怖の支配・供犠・「正しい埋葬」を巡る争いの源流。
- IF: 死を畏れ過ぎ=墓と儀式に資源を注ぐ「死に憑かれた文化」。意味づけ無=喪も愛着もない冷たい人類。
- 軸: 精神++ / 平和+ / 技術-
- ジレンマ: 悼む心は愛と倫理の源にして、見えぬものへの恐怖と支配の源。

### 7. 出アフリカと世界拡散（主要拡散 約6.5万〜5万年前）
- 岐路: 故郷は人口過密。地平線の彼方に未知の大陸、だが危険。
- 授けもの: [海を渡れ/北へ(拡散)] / [故郷を耕せ(定住)] / [少数だけ送れ]
- 諸刃: 全大陸へ=生存保険・文化分岐。だが在来大型動物相とネアンデルタール等近縁人類が消えた。
- IF: 停止=一度の災害で全滅しうる脆い世界。急ぎ過ぎ=準備不足の小集団が次々全滅。
- 軸: 技術+ / 自然-- / 精神+
- ジレンマ: 種を守る唯一の道が、他の種と他の人類を滅ぼす道。

### 8. 巨大動物相の絶滅（約5.2万〜1.1万年前, 178種超消失）
- 岐路: 巨獣一頭で群れが数週間食える。だが狩りは命がけ。
- 授けもの: [投げる武器を] / [罠と火で追え] / [小さな獲物で足りる]
- 諸刃: 高栄養・人口増。だが巨大動物相崩壊、生態系は二度と戻らず、後の農耕への遠い圧力に。
- IF: 過剰効率=饗宴のあとの飢え。狩らせない=マンモスと暮らす「もう一つの地球」(魅力的な歪み)。
- 軸: 技術+ / 自然-- / 平和-
- ジレンマ: 目の前の飢え vs 取り返しのつかない喪失。

### 9. 犬の家畜化（約1万4千年前）★最も意外なジレンマ
- 岐路: オオカミは脅威で競争相手。だが火の周りに大人しい個体が現れ始めた。
- 授けもの: [火を分け与えよ(手なずけ)] / [警戒し追い払え] / [狩って食べよ]
- 諸刃: 人類最古の相棒。だが「他種を人の都合で作り変える」最初の一歩=後の家畜・牧畜、そして人による人の支配の遠い前例。
- IF: 極端=全てを「使えるか」で見る冷たい支配者文明。結ばれず=温もりなき孤独で攻撃的な種。
- 軸: 精神+ / 技術+ / 平等-
- ジレンマ: 種を超えた友情の始まりが、支配の始まりでもある。最初の友は最初の家畜。

## 第2幕：文明の誕生（前1万年 〜 前500年）

### 10. 農業革命（前1万〜8000年, 各地で独立）★主題の提示
- 岐路: 氷期明けで獲物減・人口増。「種を蒔けば実る」気づき。
- 授けもの: [種をまく知恵] / [群れを囲う] / [実りの暦(待つことを教える)]
- 諸刃: 備蓄→定住→人口爆発。だが単一作物依存で飢饉に脆く、栄養低下、密集と家畜から疫病、余剰の管理者=格差、労働は過酷化。
- IF: 庭園採集に留める=健康で平等だが何千年も足踏み。単一作物強制=一度の不作で文明圏ごと餓死。
- 軸: 技術+ / 自然- / 平等- / 精神+
- ジレンマ: 「進歩」の最初の一手が、格差・疫病・重労働の母。神の善意が呪いになる主題。

### 11. 動物の家畜化（ヒツジ/ヤギ 前9000年〜, ウマ 前5000〜4000年）
- 岐路: 狩りは不安定、耕す力も乳も足りない。
- 授けもの: [狼を友に] / [群れを飼う] / [馬に跨る]
- 諸刃: 乳・毛・力・肥料・運搬。だが家畜から人獣共通感染症(天然痘等)=後の世界的疫病の源。馬は交易と騎馬戦・略奪を加速。
- IF: 馬だけ極端=奪う遊牧戦士文明。家畜皆無=疫病に強く穏やかだが発展鈍い。
- 軸: 技術+ / 自然- / 平和- / 精神+
- ジレンマ: 便利さの代償が「数千年後」に発火する遅延型ジレンマ。

### 12. 土器と織物（土器は農業より古い, 中国 約前1.8万年/縄文 約1.3万年前）
- 岐路: 貯める器も、加工しやすい衣もない。
- 授けもの: [火で固める土] / [糸を績む] / [文様を刻む(所有と所属)]
- 諸刃: 長期備蓄と加熱調理、衣服・帆・社会的地位。だが「貯蔵できる富」=偏在・略奪・専業職人と非生産者の分離。
- IF: 文様強調=生まれ村の模様から逃れられないカースト的世界。土器無し=定住が崩れ永遠の移動民。
- 軸: 技術+ / 精神+ / 平等- / 自然±
- ジレンマ: 地味な「器」が格差の起点。技術は必要から生まれるとは限らない。

### 13. 灌漑と治水（約6000年前〜, 塩害 前2000年頃〜）
- 岐路: 雨少ない大河流域。氾濫と渇水。一人では大河を御せない。
- 授けもの: [水路を掘れ] / [堤と溜め池(ナイル式)] / [みなで掘る掟(共同労働)]
- 諸刃: 砂漠を穀倉に=都市の母体。共同治水が中央権力・官僚・徴用を生む。長年の灌漑で塩害→収量減→耕作放棄(環境破壊による自滅)。
- IF: 大運河=水を握る者が神に等しい「水の専制」。塩害無視=数百年後に塩を吹いた死の大地。
- 軸: 技術+ / 自然- / 平等- / 平和±
- ジレンマ: 今すぐの豊作 vs 数百年後の塩害。治水という善が専制を呼ぶ皮肉。

### 14. 都市の誕生（ウルク 前4000年, インダス 前2600年）★最高の二択
- 岐路: 村が膨らみ過ぎ、顔見知りの掟では捌けない。
- 授けもの: [壁の中へ集え] / [碁盤の街(インダス式下水)] / [広場と神殿]
- 諸刃: 分業・交易・文字・行政の加速装置。だが密集=疫病・犯罪、富集中=階層社会と城壁(戦争の標的)。インダスは無王・衛生・比較的平等の稀有な例。
- IF: インダス式平等都市=清潔な技術都市だが求心力を欠き外圧に解体する儚い理想郷。巨大化のみ=疫病と暴動で内から腐る「崩れる塔」。
- 軸: 技術+ / 平等-(インダスなら±) / 精神+ / 平和- / 自然-
- ジレンマ: ウルク型(中央集権)かインダス型(無王の平等)か。正しすぎる選択が史実で謎に滅んだ不気味さ。

### 15. 文字の発明（楔形文字 前3400〜3100年, 甲骨文字 前1300年頃）
- 岐路: 倉庫の在庫も税も記憶では追えない。約束も法も人が死ねば消える。
- 授けもの: [粘土に刻む数(会計)] / [神々の言葉(占い/神官独占)] / [物語を綴る]
- 諸刃: 知識・法・歴史・契約の蓄積=文明の記憶装置。だが読める者=書記・神官の特権階級、徴税・支配・宣伝の道具に。
- IF: 神官独占=「解読されぬ真実」にひざまずく神権政治。文字無し=口承のみで同じ過ちを繰り返す。
- 軸: 技術+ / 精神++ / 平等- / 平和±
- ジレンマ: 最も与えたい純粋な善(知識)が、最強の支配ツールになる。

### 16. 組織化された宗教（ギョベクリ・テペ 前9500年〜＝宗教が文明に先行?）
- 岐路: バラバラの精霊を恐れ、見知らぬ者を束ねる共通の意味がない。
- 授けもの: [聖なる場を築け] / [神官に取り次がせる] / [捧げものの掟]
- 諸刃: 見知らぬ大集団を同価値で束ね、巨大事業と道徳を共有(社会の接着剤)。だが神殿が富と権力を吸い、異端排除・人身供犠・聖戦の種。
- IF: 神官に権力集中=教義に逆らう村を焼く恐怖政治。供犠極端化=豊作のたび祭壇が血に染まる暗黒。
- 軸: 精神++ / 平和± / 平等- / 技術+
- ジレンマ: 団結の聖なる力と、神の名の虐殺が同じ源泉。宗教が文明を生んだか、文明が宗教を生んだか。

### 17. 法と国家・官僚（ウル・ナンム法典 前2100年, ハンムラビ 前1754年）
- 岐路: 揉め事が有力者の気分次第で裁かれ、恣意的な裁きに民が怯える。
- 授けもの: [目には目を(明文化)] / [弱者を守る掟(社会改革)] / [王は神の代理]
- 諸刃: 予測可能な秩序・契約=大規模協力。だが身分差別を制度化(階級で命の値段が違う)、官僚は税・徴用・監視も強化。
- IF: 平等法極端=活力を失い停滞して呑まれる脆い福祉国家。「目には目を」極端=些細な罪も死刑の監視社会。
- 軸: 平和+ / 平等- / 技術± / 精神+
- ジレンマ: 秩序のために不平等を「正義の名の下に」永久固定するか。

### 18. 青銅器から鉄器へ（青銅 前3300年〜, 鉄 ヒッタイト 前1500年頃）
- 岐路: 石や銅はもろい。スズは産地が限られ交易必須。
- 授けもの: [銅にスズを混ぜよ(青銅/交易依存)] / [炎を極めよ(鉄/どこでも採れる)] / [鍛冶を秘伝に(独占)]
- 諸刃: 農具・工具・建築の飛躍。だが武器の殺傷力増、戦争の大規模化・職業化。鉄は武力を「民主化」し小国も軍を持て戦乱拡大。
- IF: 製鉄独占=鉄の帝国が周囲を蹂躙。武器化禁止=防御も鈍り蛮族に無防備。
- 軸: 技術++ / 平和-- / 平等±(鉄は分散) / 自然-(伐採・採掘)
- ジレンマ: 鋤にも剣にもなる火と金属。安価な力は平和をもたらさない。集中か拡散か。

### 19. 暦と天文（エジプト太陽暦 前2000年頃〜）
- 岐路: 蒔き時も氾濫も読めず、社会の予定が噛み合わない。
- 授けもの: [星で年を測れ(太陽暦)] / [月を数える(太陰太陽暦)] / [天に未来を読む(占星術)]
- 諸刃: 農事・祭祀・行政を時間で同期。天文観測=数学・予測科学の母胎。だが知識を神官・王が独占、占星術=運命論で人を縛る。
- IF: 占星術極端=全てを星に決めさせる「星に操られる世界」。観測科学に振り切り=神秘を失い信仰(社会の接着剤)が弱る。
- 軸: 技術++ / 精神+ / 平等- / 自然+
- ジレンマ: 科学(予測・制御)か神秘(信仰・権威)か。「賢さ」と「魂」のどちらを伸ばすか。

## 第3幕：古典文明〜中世（前500年 〜 西暦1500年）

### 20. 枢軸時代の思想爆発（前800〜前200年）★通奏低音
- 岐路: 各地で同時に「人はいかに生きるべきか」が噴出。神話と掟では人心をまとめられない。
- 授けもの: [理性のロウソク(ギリシャ哲学)] / [慈悲のさとり(仏陀)] / [礼と仁の秩序(孔子)] / (隠し)[善悪二元の火(ゾロアスター)]
- 諸刃: 内省と普遍倫理=全宗教・哲学・科学の土台。だが「正しさ」が教義化し異端排除の論理も生まれた。
- IF: 二元論強調=「光の民/闇の民」に分断され永遠の聖戦。内省過剰=現世を軽視し誰も橋を直さない「考えるだけの文明」。
- 軸: 精神+++ / 平等+(思想は身分を超える)/-(教団内序列) / 平和±
- ジレンマ: 思考の自由 と 社会の結束 は逆を向く。賢くなるほどまとまりにくい。

### 21. 巨大帝国の統治インフラ（道路・官僚・文字統一, 前500〜後200年）
- 岐路: 領土が広すぎ、王の命令が辺境に届かない。「自分の重さ」で崩れかけ。
- 授けもの: [王の道(街道)] / [試験で選ぶ官(漢の官僚制)] / [公の手紙網]
- 諸刃: 平和・交易・法・郵便を辺境へ(「ローマの平和」)。同じ道が軍団と徴税の道具に。試験制は実力登用だが暗記偏重で革新を抑制。
- IF: 官僚制過剰=全てが書類で動き「前例なし」で却下される静止帝国。道だけ=盗賊と疫病の高速道路。
- 軸: 技術++ / 平和+(域内)/-(征服) / 平等+(登用)/-(集権) / 自然-
- ジレンマ: 効率的統治の道具は善政にも圧政にも等しく使える。

### 22. 世界宗教の成立と拡大（仏教 前3世紀/キリスト教 313年/イスラム 622年）
- 岐路: 多民族の帝国に共通の心の絆がなく、疫病・戦乱で救いを渇望。
- 授けもの: [菩薩の旅(アショーカ)] / [皇帝の十字(コンスタンティヌス)] / [最後の預言(イスラム)]
- 諸刃: 血や言語を超え人を「兄弟」に=慈善・病院・教育・国際学問網。だが正典化の過程で異端審問・宗教戦争・改宗強制。
- IF: 一神を国家の旗に=神権政治が暴走し科学者も火あぶり。三宗教同時=決着しない聖地に全文明が軍と巡礼を送り続ける。
